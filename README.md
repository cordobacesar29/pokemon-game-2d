
Pokémon Game 2D 🎮

Un juego de rol (RPG) estilo 2D inspirado en la estética clásica de Pokémon, desarrollado íntegramente en Java. Este proyecto demuestra el uso de gráficos 2D, manejo de eventos de teclado y lógica de colisiones.

✨ Características
Motor 2D Propio: Implementación de un ciclo de juego (Game Loop) para actualizaciones y renderizado constantes.

Movimiento del Jugador: Sistema de desplazamiento en cuatro direcciones (Arriba, Abajo, Izquierda, Derecha).

Manejo de Sprites: Renderizado de gráficos para el personaje y el entorno.

Mapa basado en Tiles: Estructura de baldosas para la construcción del escenario.

🛠️ Tecnologías Utilizadas
Lenguaje: Java ☕

Interfaz Gráfica: Java Swing / AWT.

Paradigma: Programación Orientada a Objetos (POO).

📂 Estructura del Proyecto
El código se organiza en paquetes para separar las responsabilidades:

main: Contiene el punto de entrada (Main.java) y el panel principal del juego (GamePanel.java).

entity: Define las entidades del juego, incluyendo la clase Player.

tile: Gestiona el sistema de mapas y texturas del suelo/obstáculos.

🚀 Instalación y Ejecución
Clonar el repositorio:

Bash
git clone https://github.com/cordobacesar29/pokemon-game-2d.git
Abrir en un IDE: Importa el proyecto en IntelliJ IDEA, Eclipse o NetBeans.

Ejecutar: Localiza la clase Main.java en el paquete main y ejecútala.

⌨️ Controles
W / A / S / D: Mover al personaje.
